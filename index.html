Data pipeline

The nightly script turns brand mentions and creator feeds into JSON the client can consume.

  • scripts/rc/build-site-data.py reads data/brand-mentions.json and data/creators/*/videos.json.
  • Generated files land in products/rc-parts-platform/data/ for brands, creators, and search.
  • The static pages load that JSON client-side so no build step is required.

Deploying this folder to GitHub Pages keeps the experience fast and free.
